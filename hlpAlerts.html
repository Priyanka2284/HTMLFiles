Introduction

Users can filter alerts by Customer, Site, Device Type, Device Name, Alert Type, Severity, and Status. The list of alerts displayed is based on the selected filters. When the filter is applied, a label Filter Applied is displayed on screen. When there is no filter applied then Default Data label is displayed on screen. Users have the options to save, clear, and delete filters. Alerts can be viewed within a selected date range. User can search the alerts using the Search Alert ID.

Saved Filters: The Saved Filters dropdown displays a list of all saved filters. Users can select any of the saved filters from the list. If no filters are saved, a message stating "No Saved Filter" is displayed. Each saved filter comes with two options:
Delete Icon: Click on the delete icon to remove the selected filter.
Info Icon: Click on the info icon to view the details of the selected filter.

Alert View Options
The Saved Filters dropdown displays a list of all saved filters. Users can select any of the saved filters from the list. If no filters are saved, a message stating "No Saved Filter" is displayed.
  • TD (Today's Alerts): Displays today's closed, open, and new alerts.
  • YD (Yesterday's Alerts): Displays yesterday's closed and open alerts.
  • 7D (Last 7 Days): Displays alerts from the last 7 days.
  • 30D (Last 30 Days): Displays alerts from the last 30 days.
